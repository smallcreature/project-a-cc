# Storage And Stockpiles

## Definition

Storage and stockpiles manage inventory pressure, preservation, item sorting, and resource buffers.

## Why It Matters

Storage creates early logistics friction, then magic can produce strong QoL by expanding, sorting, preserving, and remotely accessing items.

## Details

Problems:

- Player inventory is limited.
- Many item types appear early.
- Food and some resources spoil.
- Many containers create sorting/search friction.

Storage conditions can extend spoilage timer:

- In container.
- Under roof.
- Inside room.

Container types:

- Baskets: primitive, small, portable, can be placed in world.
- Chests: standard larger container, not portable.
- Backpacks: portable, smaller than chests.
- Crates: huge capacity, one item type.
- Buckets: one unit of any liquid.
- Barrels: larger liquid container, one liquid type.

Magic can:

- Increase container capacity.
- Give remote access.
- Add personal containers.
- Extend preservation.
- Allow nesting some containers.
- Auto-unload player inventory.
- Sort items into matching containers.
