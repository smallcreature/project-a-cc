# Deep Co-op

## Definition

Deep co-op means long, specialized progression branches create meaningful roles in small groups.

## Why It Matters

Project hypothesis: engagement and session length rise when players can divide work by preference and feel stronger together than alone.

## Details

Potential roles:

- Resource gatherer / miner.
- Explorer / dungeon runner.
- Farmer / cook.
- Researcher.
- Builder / infrastructure planner.
- Combat specialist.
- Magic school specialist.
- Logistics/automation specialist.

Co-op is strengthened by:

- Many magic schools.
- Separate non-magic specializations.
- Shared base infrastructure.
- Resource logistics.
- Bosses, dungeons, PvE raids.
