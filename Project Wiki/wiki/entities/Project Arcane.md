# Project Arcane

## Definition

Project Arcane is a survival-craft game where the player is a scientist who has just discovered magic. The player must learn how magic works through [[concepts/Immersive Research|research]], resource experiments, exploration, and repeated application.

## Why It Matters

The project fantasy is transformation: vulnerable survivor -> capable researcher -> specialized mage -> near-godlike being able to bypass core survival-craft constraints.

The game is built around layered systems. Early play emphasizes manual labor, needs, storage, crafting, and local resources. Later play uses magic to reduce friction, automate routine work, reach dangerous content, and alter the world.

## Tensions / Contradictions

- The game wants both immersive complexity and accessibility. Systems must be deep for engaged players but forgiving for casual players.
- Magic must feel powerful enough to break rules, but its world consequences must not make progress feel punitive.

## Open Questions

- What is the exact MVP slice: which schools, systems, bosses, and automation features ship first?
- How much of the "researcher fantasy" depends on hidden information versus explicit tutorials?
